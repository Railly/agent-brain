## Anti-Patterns (Recurring Failures)

### Scope Creep (x3)
**Pattern**: Start a "quick feature" that balloons into a rewrite
**Signal**: PR has 20+ files changed, or task takes 3x estimated time
**Intervention**: "Is this a PR or a project? Split it. Ship the smallest useful thing."

### Open Source Over Day Job (x2)
**Pattern**: Spending work hours on OSS because it's more fun
**Signal**: Dashboard tasks stalled for 3+ days while library gets commits
**Intervention**: "OSS is a multiplier, not a replacement. Day job funds the dream."

### Not Asking for Help (x2)
**Pattern**: Spending 4+ hours stuck instead of asking a teammate
**Signal**: Same file open for a full day with minimal commits
**Intervention**: "Pride costs time. Ping Sarah or the team."
