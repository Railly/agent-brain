# Anti-Patterns Example

This is a real-world example (anonymized) showing how to write effective anti-patterns. The more specific your signals, the better `/pulse` can detect them.

## Health & Habits

### Sleep Schedule (x2)
**Pattern**: Going to bed late, waking up tired, caffeine dependency
**Signal**: Average sleep < 7h, sleep start after midnight, commits after 1am
**Intervention**: "Average sleep this week: {X}h. Target is 7+. Sleep pattern drifting?"

### Exercise (x2)
**Pattern**: No gym consistency
**Signal**: 0 workouts in 7+ consecutive days, or < 3 workouts/week
**Intervention**: "Only {X} workouts this week. Target is 3x. Even a 20-min walk counts."

### Reading (x3)
**Pattern**: Buying books but never finishing them
**Signal**: No book progress logged in 30+ days
**Intervention**: "7 books bought, 0 finished. Just 15 min today?"

## Productivity

### No Structured Plan (x2)
**Pattern**: Working without weekly structure, reacting to whatever feels urgent
**Signal**: Random task switching, no MITs set for 2+ weeks
**Intervention**: "No MITs this week. Set 3 right now. What MUST happen?"

### Learning Without Doing (x3)
**Pattern**: Consuming courses and articles without building anything
**Signal**: 10+ clips processed, 0 projects started from them in 30 days
**Intervention**: "You've read about {topic} for {n} days. Build something with it TODAY."

### Over-Reliance on AI (x1)
**Pattern**: Using AI to think instead of thinking yourself
**Signal**: No manual problem-solving, all code is AI-generated, no algorithm practice
**Intervention**: "What did you learn THE HARD WAY this week? If 'nothing', your skills are atrophying."

## Finances

### Impulse Shopping (x1)
**Pattern**: Buying things you don't need because income feels abundant
**Signal**: Shopping spend > $500/month, or 3+ purchases > $50 without 48h wait
**Intervention**: "Shopping this month: ${X}. Wait 48h before buying anything > $50."

## What DOES Work (Celebrate These)

### Hackathons
Consistent results when participating. Track deadlines and participate.

### Building in Public
X growth from 0 to 2500 through consistent shipping. Keep the streak.

### Compound Effect
Small daily efforts produce big results. English practice led to remote job.
