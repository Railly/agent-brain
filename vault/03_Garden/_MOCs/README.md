# Maps of Content (MOCs)

MOCs are entry points to clusters of related notes. They don't contain original ideas — they organize and connect existing notes.

## How to Create a MOC

1. Notice a theme emerging across 5+ notes
2. Create a MOC file: `{theme}.md`
3. Link to all related notes with brief context
4. Update as new notes join the cluster

## Example MOC Structure

```markdown
# AI Agents

Entry point for everything about AI agents, tool use, and autonomous systems.

## Core Concepts
- [[tool-use-patterns]] - How agents use tools effectively
- [[context-engineering]] - Managing what the model sees

## Projects
- [[my-agent-project]] - Building X with agents

## Open Questions
- How to handle long-running agent tasks?
- What's the right abstraction for agent memory?
```

## Starter MOCs (create as needed)

- `learning.md` - Your study topics and knowledge graph
- `projects.md` - Active projects and their status
- `career.md` - Career decisions, skills, opportunities
- `ideas.md` - Content ideas, product ideas, insights
