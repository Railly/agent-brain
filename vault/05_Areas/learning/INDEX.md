# Learning System

Track study sessions, build knowledge, practice retrieval.

## Commands
- `/prime {topic}` - Pre-lesson briefing (3 min scaffolding)
- `/learn {topic}` - Create study note with review questions
- `/recall` - Morning retrieval practice quiz
- `/study {topic} {duration}` - Log a study session

## Flow
```
/prime → lesson → /learn → next day → /recall → repeat
```

## Targets
- 25 days/month with study activity
- AM: 15min recall practice
- PM: 45min new material
- At least 3 topics interleaved per month

## Study Sessions
Logged in `daily/YYYY-MM-DD.md` with duration, topic, method, and self-generated questions.
